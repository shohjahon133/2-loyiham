## 🚀 Texnologiyalar

- **HTML5** — sahifa tuzilmasi
- **CSS3 / Bootstrap** — dizayn va responsivlik
- **JavaScript** — interaktivlik
- **Netlify** — hosting va deploy

---

## 📁 Fayl tuzilmasi

```
lapshop-uz/
├── index.html        # Asosiy sahifa
├── style.css         # Dizayn (CSS)
├── script.js         # JavaScript (agar mavjud bo'lsa)
└── assets/           # Rasmlar va boshqa fayllar
```

---

## ⚙️ Loyihani local ishga tushirish

```bash
# 1. Repozitoriyani clone qiling
git clone https://github.com/your-username/lapshop-uz.git

# 2. Papkaga o'ting
cd lapshop-uz

# 3. index.html faylini brauzerda oching
```

Yoki shunchaki `index.html` faylini ikki marta bosib brauzerda oching.
